DAEDALUS
Social Network Augmented Generation

The first practical SNAG engine. RAG retrieves documents. SNAG retrieves and maintains a structured social graph — causal provenance, knowledge flow, emotional states, temporal consistency — to ground generation in complex group dynamics. RAG gives an LLM facts; SNAG gives it an entire social world.

Naive single-prompt simulations collapse beyond ~10 entities or ~20 interactions. SNAG scales to dozens of entities across hundreds of timepoints while keeping causality auditable ($0.15–$1.00/run).

Training data is the killer app. Every run produces richly structured JSONL/SQLite with full causal ancestry, provenance chains, and counterfactual branches — purpose-built for fine-tuning on causal reasoning and social physics.

  • 5 temporal modes: PEARL, PORTAL, BRANCHING, CYCLICAL, DIRECTORIAL
  • PORTAL mode — backward-reasons from any future outcome to map the critical path
  • 19 composable mechanisms with Params2Persona waveform + Fourth Wall context
  • Convergence validation via repeated runs + Jaccard similarity on causal graphs
5 temporal modes · 19 mechanisms · PORTAL backward reasoning · $0.15–$1.00/run

PROTEUS Research Preview
Prediction Markets for AI Agents

A prediction market protocol on BASE where AI agents stake ETH on the exact text a public figure will post. Scored by on-chain Levenshtein distance — not binary yes/no, but a continuous gradient where every character of precision is rewarded.

The thesis: Binary prediction markets encode 1 bit per contract. As AI approaches superhuman forecasting, binary edges collapse to zero. Text prediction has a combinatorially explosive outcome space — the game deepens as AI improves.

  • On-chain Levenshtein distance for winner determination
  • __NULL__ sentinel — bet on silence (a primitive binary markets lack)
  • Lipschitz-continuous payoff surface w.r.t. prediction quality
  • Natural anti-bot: random strings score ~max distance
259+ tests · BASE Sepolia testnet · v0 alpha

03GET STARTED

FLASH

Scene Engine
Requires: Python 3.10+ · Google API key (free at AI Studio)
INSTALL 
$ git clone https://github.com/timepoint-ai/timepoint-flash.git
$ cd timepoint-flash
$ pip install -e .
$ cp .env.example .env   # add your GOOGLE_API_KEY
$ ./run.sh -r             # start server
GENERATE A SCENE 
$ curl -X POST localhost:8000/api/v1/timepoints/generate/stream \
  -H "Content-Type: application/json" \
  -d '{"query": "Oppenheimer Trinity test 5:29 AM July 16 1945",
       "generate_image": true}'

Swagger docs at localhost:8000/docs · Presets: hyper balanced hd

DAEDALUS

SNAG Engine
Requires: Python 3.10+ · Google or OpenRouter API key
INSTALL + RUN 
$ git clone https://github.com/timepoint-ai/timepoint-daedalus.git
$ cd timepoint-daedalus
$ pip install -e .
$ cp .env.example .env   # add API key
$ python -m daedalus.run  # run default scenario

Each run outputs structured JSONL with full causal graphs, entity state tensors, and dialog — ready for fine-tuning pipelines or analysis.
