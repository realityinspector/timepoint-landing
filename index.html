Daedalus

Social simulation at scale. The first SNAG engine.

Social Network Augmented Generation. Like RAG retrieves documents, SNAG synthesizes and maintains a structured social graph — causal provenance, knowledge flow, emotional states, temporal consistency — to ground LLM generation in group dynamics. Run a Mars mission backward from failure to find the institutional cracks. Branch a colonial survival scenario into counterfactuals. Render social systems like a synthesizer renders sound: variable fidelity, any scale.

Proteus

Prediction markets that reward precision, not just conviction.

Binary prediction markets encode one bit of information per contract. As AI forecasting approaches superhuman calibration, the edge any participant can capture collapses toward zero. Proteus scores predictions by Levenshtein edit distance — exact text prediction over a continuous gradient surface where every character of precision is rewarded. 10554 possible outcomes per market versus two. The game deepens as models improve.

Solidity on BASE · 259+ tests · On-chain Levenshtein · v0 alpha
